Simply put- What is Javascript?

Javascript is a scripting language often considered the language of the internet.
It is versatile and heavily used in web and app development.
A key feature that sets Javascript apart is its ability to interact with the browser DOM which is used in the construction of a dynamic web page.

Javascript uses

It stands apart from HTML and CSS in the way it is used to manipulate and drive events and dynamic changes within its environment, there are not many similarities between Javascript and the others- however they work in unison. Without any one of these the internet would not be the place it is today.
